This site was designed and engineered on a MacBook Pro using Visual Studio Code for coding, Webpack for compiling SASS and minifying JavaScript, plus the Adobe Creative Suite for creating design assets.

Pages on this site are generated by Jekyll and served by Apache running on a Digital Ocean server.
